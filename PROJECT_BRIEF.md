# FootyBrain Project Brief

FootyBrain is a soccer/football learning database with quiz elements.

Core idea:
Users can browse leagues, teams, and players. The app helps them learn player names, positions, current clubs, national teams, career history, quick facts, and team/fan history.

Tagline:
Learn the game. Know the players. Become a real fan.

MVP goals:
- Frontend-only React + Vite app
- No external soccer API yet
- Use local sample data
- Browse by league and team
- Search players
- Player cards
- Player profile detail view
- Team learning pages
- Quiz mode with hints
- Mobile-friendly design

Important:
- Do not use copyrighted EA FC ratings.
- Use our own “FootyBrain Rating” or “Importance Score.”
- Keep data in local sample files for now.
- Do not add Firebase yet.
- Do not connect real APIs yet.
- Keep code clean and easy to expand.

Main pages/components:
- Home
- BrowseDatabase
- PlayerCard
- PlayerProfile
- TeamLearning
- TeamCard
- QuizMode
- Navbar
- sampleData

Design vibe:
Clean, modern sports database. Not cluttered. Easy to use on phone.
